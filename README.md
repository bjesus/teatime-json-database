# Teatime JSON Database

This is an example [Teatime](https://github.com/bjesus/teatime/) database that is automatically generated from [data.json](data.json). It uses GitHub Actions to create an SQLite file and then publishes it to GitHub Pages.

Forking this database and adjusting data.json and config.json to the files you would like to serve using Teatime is the probably the easiest way to create a Teatime database.

You can make your files available in IPFS using services like [Pinata](https://pinata.cloud) or by running an IPFS node locally.
